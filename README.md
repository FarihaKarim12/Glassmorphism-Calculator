# Glassmorphism Calculator

A modern calculator built with HTML, CSS, and JavaScript featuring a frosted glass UI design.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Square root (√) and percentage (%) support
- Calculation history panel with click-to-reuse
- Dark / Light mode toggle
- Full keyboard support
- Responsive design (desktop, tablet, mobile)

## Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `0–9` | Input numbers |
| `+ - * /` | Operators |
| `Enter` | Calculate |
| `Backspace` | Delete last digit |
| `Escape` | Clear |
